# Component Specifications

## Arduino UNO R3
- **Microcontroller**: ATmega328P
- **Operating Voltage**: 5V
- **Input Voltage**: 7-12V
- **Digital I/O Pins**: 14
- **Analog Input Pins**: 6
- **Clock Speed**: 16 MHz
- **Flash Memory**: 32 KB
- **SRAM**: 2 KB
- **EEPROM**: 1 KB

## MFRC522 RFID Module
- **Operating Frequency**: 13.56 MHz
- **Operating Voltage**: 3.3V
- **Current Consumption**: 13-26mA
- **Read Range**: 0-60mm
- **Interface**: SPI, I2C, UART
- **Data Transfer Rate**: 106-424 kbps
- **Crypto Engine**: Yes
- **Operating Temperature**: -20°C to +85°C

## 2-Channel Relay Module
- **Operating Voltage**: 5V DC
- **Current Consumption**: 70mA per channel
- **Relay Type**: Electromechanical
- **Max Switching**: 250V AC / 30V DC
- **Max Current**: 10A
- **Contact Resistance**: 100mΩ max
- **Operate Time**: 10ms max

## Electric Strike Lock
- **Operating Voltage**: 12V DC
- **Current**: 250-500mA
- **Lock Type**: Fail-Secure
- **Holding Force**: 1200 lbs
- **Faceplate**: Stainless Steel
- **Operating Temperature**: -20°C to +65°C
- **Duty Cycle**: Intermittent

## Power Supply
- **Input**: 100-240V AC, 50/60Hz
- **Output**: 12V DC, 2A
- **Efficiency**: >85%
- **Regulation**: ±5%
- **Protection**: Short Circuit, Over Voltage
- **Safety Standards**: UL/CE/RoHS
